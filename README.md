
# Reinforcement-Learning-Game-AI

[![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)](https://www.python.org/)
[![Reinforcement Learning](https://img.shields.io/badge/Reinforcement%20Learning-F7931E?style=for-the-badge&logo=n-a&logoColor=white)](https://en.wikipedia.org/wiki/Reinforcement_learning)

Implementing various Reinforcement Learning algorithms to train AI agents for simple game environments.

## Overview

This repository explores the application of Reinforcement Learning (RL) to develop intelligent agents capable of playing and mastering simple game environments. It provides implementations of fundamental RL algorithms and demonstrates their effectiveness through practical examples.

## Features

-   **Q-Learning**: Implementation of the Q-learning algorithm.
-   **SARSA**: Implementation of the SARSA algorithm.
-   **Environment Integration**: Examples of integrating RL agents with OpenAI Gym-like environments.
-   **Visualization**: Tools for visualizing agent learning progress and policy.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Python 3.8+
-   pip

### Installation

1.  Clone the repo
    ```bash
    git clone https://github.com/Constainey5/Reinforcement-Learning-Game-AI.git
    ```
2.  Install Python packages
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the example scripts in the `src/` directory to train and evaluate RL agents.

```bash
python src/q_learning_frozen_lake.py
```
